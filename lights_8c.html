Detailed Description

Control de brillo de iluminación en base a luz ambiente con media móvil y PWM.

Este archivo implementa las funciones necesarias para leer un sensor de luz (fotocelda), aplicar una media móvil para estabilizar la señal y ajustar el brillo de una fuente de luz mediante modulación PWM. Se utiliza el canal ADC 1 (GPIO27) para la lectura del sensor.

El duty cycle se adapta en tiempo real según la cantidad de luz ambiente detectada. PWM configurado a 10 kHz para evitar parpadeos perceptibles.
